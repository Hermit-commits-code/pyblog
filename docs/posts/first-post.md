# First Post — Getting Started

Welcome to the first post on PyBlog. This site uses MkDocs + Material theme and is deployed automatically to GitHub Pages.

This is a minimal, fast, and modern static blog that you can update by adding Markdown files under `docs/`.

Next steps:

- Write posts in `docs/posts/`.
- Add images under `docs/images/` and reference them with relative paths.
- Use the GitHub UI or `git` to push and the site will auto-publish.
